# Front-end Style Guide

## Layout

The designs were created to the following widths:

- Mobile: 375px
- Desktop: 1440px

## Colors

### Primary

- Moderate cyan: hsl(176, 50%, 47%) hex #3cb4ac
- Dark cyan: hsl(176, 72%, 28%) hex #147b74

### Neutral

- Black: hsl(0, 0%, 0%) hex #000000
- Dark gray: hsl(0, 0%, 48%) hex #7a7a7a
